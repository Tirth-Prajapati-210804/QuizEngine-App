# Quiz Engine

## About the Project

Quiz Engine is a command-line based quiz application developed using Python.
It allows users to add quiz questions and attempt quizzes stored in a database.

This project focuses on logic building, database interaction,
and writing modular and readable code.

## Features

- Add quiz questions
- Attempt quizzes
- Automatic score calculation
- Database-backed question storage

## Technologies Used

- Python
- SQLAlchemy (ORM)
- SQLite Database

## How to Run the Project

1. Install Python
2. Open terminal in the project directory
3. Run the command:

python main.py

Quiz data will be saved automatically in the database.
